# Network Labs

Cisco Packet Tracer labs demonstrating core networking concepts through small, hands-on simulations.

## Labs

- [01-home-network](01-home-network/) - Simple home network with DHCP, IPv4 addressing, wireless connectivity, and connectivity testing.
- [02-wireless-router-dhcp](02-wireless-router-dhcp/) - Wireless router and DHCP configuration.
- [03-nat-analysis](03-nat-analysis/) - NAT behavior and traffic analysis.
- [04-create-a-lan](04-create-a-lan/) - Local area network design and configuration.
- [05-routed-network-traffic](05-routed-network-traffic/) - Routed network traffic flow.
- [06-ssh-and-telnet](06-ssh-and-telnet/) - Remote access using SSH and Telnet.

## Tools

- Cisco Packet Tracer
- IPv4 addressing
- DHCP
- Ping and ipconfig
- Basic troubleshooting workflows
